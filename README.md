# sky

#### 介绍

- 基于 Spring Cloud Hoxton.SR5 、Spring Boot 2.3、 OAuth2 的RBAC权限管理系统

- 基于 VUE 的组件库 element-ui 实现前后端分离的微服务架构

- 采用 Nacos 作为注册中心、配置中心


#### 代码结构


```
|sky------------------------------------------------  项目名
    ├─doc--------------------------------------------  初始化脚本
    ├─sky-auth---------------------------------------  认证中心
    ├─sky-common-------------------------------------  公共类
    ├─sky-gateway------------------------------------  网关程序入口
    ├─sky-monitor------------------------------------  监控
    ├─sky-system-------------------------------------  业务
        ├─sys-generator-------------------------------  代码生成工具
        ├─sys-sys-------------------------------------  系统设置
```
